Preferences


Website: Select the website to display in the top portion of StockTracker. The default is the Yahoo Finance home page, but this may be changed to any website (e.g., https://finviz.com/) that is useful for seeing updates to market indices and other stock information.

API Key: StockTracker relies on the Finnhub stock API for updating stock prices and retrieving news. Please go to the Finnhub website and click on the green "Get free API key" button to request a free API key. Then enter the 40 digit key here.

Update every n minutes: Enter the number of minutes (5 - 60) between stock price and news updates.
